Water Innovation – Milton Roy Pump Enquiry Portal

This repository hosts a lead generation enquiry form designed for collecting technical requirements for Milton Roy dosing pumps used in industrial applications such as Water Treatment Plants (WTP), Effluent Treatment Plants (ETP), Chemical Processing, and Pharmaceutical industries.

The web interface allows visitors to submit pump requirements including model selection, flow rate, pressure rating, and chemical dosing application.

The project is hosted using GitHub Pages and provides a simple interface for industrial enquiry collection.

Project Purpose

The objective of this project is to provide a simple and accessible enquiry system for customers who require Milton Roy dosing pump solutions.

It helps collect essential technical information required for pump selection and quotation.

Features

Industrial enquiry form for dosing pump requirements

Milton Roy pump model selection

Flow rate input (LPH)

Pressure requirement input (Bar / Kg/cm²)

Chemical dosing information

Contact information collection

Email enquiry submission to support team

Lightweight static website compatible with GitHub Pages hosting

Technologies Used

HTML5

CSS3

JavaScript

GitHub Pages (Static Hosting)

Deployment

This website is deployed using GitHub Pages.

To deploy the project:

Clone or download this repository

Ensure the main webpage is named index.html

Push the project to a GitHub repository

Enable GitHub Pages in repository settings

Select the main branch as the source
